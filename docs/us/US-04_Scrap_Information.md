User Story

As a user
I want to compare information from different external web sources
so that I can enrich my fossil records with reliable data.

Acceptance Criteria

 The user can input a species or topic to trigger web comparisons.

 The system fetches information from multiple predefined websites/APIs.

 The comparison displays differences/similarities between sources.

 The user can choose to save selected information into a record.

 If external sources fail, a clear error message is shown.

 Only publicly available and safe data is retrieved.
