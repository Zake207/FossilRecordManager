User Story

As a user
I want to edit or delete my records
so that I can keep my information accurate and up to date.

Acceptance Criteria

 Only the owner of the record can modify or delete it.

 Editing allows updating text fields, species info, and images.

 Deleting a record removes all associated images and data.

 A confirmation step is required before deletion.

 Changes are saved in the database and reflected immediately.

 The user receives visual confirmation of edits or deletions.
