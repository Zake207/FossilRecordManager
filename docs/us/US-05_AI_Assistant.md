User Story

As a user
I want to ask questions to an AI assistant
so that I can obtain explanations, summaries, or insights about species.

Acceptance Criteria

 Users can submit questions related to species or fossil records.

 The AI assistant responds with clear, relevant explanations.

 The assistant can suggest related species or terms.

 The user can optionally add AI-generated content to a record.

 The system logs AI queries to improve context in future interactions.

 If the question is unclear, the assistant asks for clarification.
