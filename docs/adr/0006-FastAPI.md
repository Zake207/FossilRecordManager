# ADR 0003: Use of FastAPI as main backend module

**Date:** 18-11-2025  
**State:** Accepted

## Context
The application need a powerful tool that allow the develop of a fast and efficient API, with scalable features and competitive status in the ambit.

## Election
The tool selected is *FastAPI* 

## Considered Alternatives
- Django
- Flask


## Consequences
### Good
- Good tool to start with
### Bad
- Kinda new, maybe dont implements some basic aspects
