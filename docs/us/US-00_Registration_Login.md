User Story

As a user
I want to register and log into my account
so that I can privately manage my fossil records.

Acceptance Criteria

 The user can create an account by providing an email and password.

 The email must be validated as unique.

 The password must meet minimum requirements (e.g., 8 characters).

 The user can log in using their registered credentials.

 Incorrect credentials trigger a clear error message.

 User sessions persist until the user logs out.
